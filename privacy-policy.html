📋 1. 概述

GPS Simulator（以下簡稱「本應用程式」）是一款 GPS 位置模擬工具，允許使用者透過 Android 開發人員選項功能，在裝置上模擬虛擬 GPS 位置。本隱私權政策說明本應用程式如何收集、使用及保護與您相關的資料。

✅ 核心承諾：本應用程式所有資料均儲存於裝置本地，不會上傳至任何外部伺服器，不收集個人身份資訊，不進行任何使用行為追蹤。

💾 2. 儲存於裝置的資料

以下資料僅儲存於您的裝置本地（Android SharedPreferences），永遠不會傳送至任何伺服器或第三方：

資料類型儲存用途
最後一次模擬的 GPS 座標（緯度、經度）下次開啟 APP 時自動還原上次位置
我的最愛地點清單（名稱、緯度、經度）快速選取常用位置
搖桿速度等級（慢／中／快）記住使用者偏好設定
GPS 注入間隔設定（毫秒）記住使用者偏好設定
搖桿最高速度（km/h）記住使用者偏好設定
介面語言設定（繁中／英文／日文）記住使用者語言偏好

🚫 3. 我們不收集的資料

  • ❌ 真實 GPS 位置（僅供畫面顯示，不儲存、不上傳）
  • ❌ 個人識別資訊（姓名、電子郵件、電話號碼等）
  • ❌ 裝置識別碼（IMEI、Android ID、廣告 ID 等）
  • ❌ 使用行為追蹤或應用程式分析資料
  • ❌ 崩潰報告或錯誤日誌（不收集、不上傳）
  • ❌ 第三方廣告或分析 SDK 資料

🌐 4. 網路使用

本應用程式使用網際網路連線的唯一目的是下載 OpenStreetMap 地圖圖磚。

  • 地圖資料由 OpenStreetMap 社群提供（openstreetmap.org）
  • 下載請求僅包含地圖座標（zoom/x/y 圖磚編號），不包含任何個人識別資訊
  • 地圖圖磚會快取於裝置本地以減少重複下載
  • 本應用程式不與任何自建後端伺服器通訊

📋 1. Overview

GPS Simulator is a GPS location spoofing tool that allows users to simulate virtual GPS positions on their Android device using the Developer Options. This Privacy Policy explains how the app handles data related to you.

✅ Core Commitment: All data is stored exclusively on your device. We never upload data to external servers, collect personal identifiers, or track usage behavior.

💾 2. Data Stored on Device

The following data is stored locally on your device only (Android SharedPreferences) and is never transmitted to any server or third party:

Joystick speed level (slow/medium/fast)Remember user preferences

Joystick max speed (km/h)Remember user preferences
Interface language (ZH/EN/JA)Remember language preference

🚫 3. Data We Do NOT Collect

  • ❌ Real GPS location (displayed on screen only, never stored or uploaded)
  • ❌ Personal identifiers (name, email, phone number, etc.)
  • ❌ Device identifiers (IMEI, Android ID, advertising ID, etc.)
  • ❌ Usage analytics or behavioral tracking
  • ❌ Crash reports or error logs
  • ❌ Third-party advertising or analytics SDK data

🌐 4. Internet Usage

The app uses the internet solely to download OpenStreetMap map tiles.

  • Map data is provided by the OpenStreetMap community (openstreetmap.org)
  • Requests contain only tile coordinates — no personal information
  • Tiles are cached locally to reduce repeated downloads
  • The app does not communicate with any proprietary backend server
